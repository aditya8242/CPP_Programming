# Run the program(s):
  1) First clone the repository on your system.
  2) Open terminal in the project directory and compile any program file using below command. 
 	 ```
	 g++ <filename>.cpp
	 ```
  3) Then
  	 ```
  	 ./a.out
  	 ```
